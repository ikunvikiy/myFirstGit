9、表数据的增  删  改  查（掌握）
添加数据 ：  insert into 表名(字段1,字段2,...) values(值1,值2,.....)    返回受影响的行数

删除数据 :   delete from 表名  清空表数据       返回受影响的行数
            delete  from  表名 where 条件
条件 ： =  表示判断  等于 
       and  表示并且
       or   表示 或者

例如 ： 删除product表中 编号为2的数据
    delete from product where pid = 2

修改数据 ： update 表名 set 字段名 = 值 , 字段名 = 值 ,... where 条件    返回受影响的行数 

例如 ：修改 编号为3的产品的价格是 10  产地是上海
       update product set price=10,paddr='shanghai' where pid=3 
      修改  产地是南京 商品编号是10 的商品 价格为100
       update product set price=100 where paddr='南京' and pid=10  

查询数据 :   
    select * from 表名  查询表中所有的数据    *  表示要查询的所有字段
    select pname , price from 表名    查询特定字段   
    select * from 表名  where 条件   给定条件查询   

例如 ：查询价格在20--100之间的商品的所有信息
        select * from product where price > 20 and price < 100
      查询价格大于30 的商品名称和产地
        select pname , paddr from product where price >30

10、主键（掌握）
主键特点 ： 数据不能重复  不能为空
一般会将一个表中的 编号 字段作为主键  
主键关键字 :  primary key
通常情况下，可以将主键设置为自增的，设置为自增后，在为表添加数据时，可以忽略主键 
自增关键字 ： auto_increament

创建一张表，并设置表编号字段为 主键  自增的
create table pro
(
    pid int primary key auto_increment,  主键自增
    pname varchar(10),
    price int,
    paddr varchar(100)
)
